COMO COLOCAR OS SCREENSHOTS
============================

Forma fácil (recomendada):
  1. Abra o ficheiro  admin-imagens.html  no browser (basta duplo-clique).
  2. Arraste cada screenshot para o espaço do projecto certo.
     Fica redimensionado para 1600x1000 e comprimido automaticamente.
  3. Reveja (clique para ampliar) e remova o que não gostar.
  4. Clique em "Descarregar todas".
  5. Copie os ficheiros descarregados para esta pasta (img/).

Feito. O portfólio mostra-as sozinho — não precisa de mexer no código.
